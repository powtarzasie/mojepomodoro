# 🍅 MojePomodoro — strona pobierania

Landing page aplikacji **MojePomodoro** — darmowego timera Pomodoro na Windows,
zawsze widocznego nad oknami, z listą zadań, importem/eksportem CSV i rejestrem
„plan vs fakt".

🔗 **Strona:** https://powtarzasie.github.io/mojepomodoro/
⬇️ **Pobierz (instalator .exe):** [Releases](https://github.com/powtarzasie/mojepomodoro/releases/latest)

---

## O aplikacji

- **Tylko Windows 10/11 (64-bit)** — aplikacja desktopowa, nie działa na telefonie ani tablecie.
- Zbudowana w technologii **Electron**; instalator zawiera wszystko — nie trzeba dograć Node.js ani mieć uprawnień administratora.
- W pełni **lokalna i offline** — bez kont, bez telemetrii. Ustawienia i log zapisują się na dysku użytkownika.
- **Open source — licencja MIT.**
